UX design

Notre travail est centré sur l'utilisateur et son univers. Votre projet sera construit autour du Responsive Web Design pour le rendre plus accessible, plus ergonomique et être présent sur tous les supports Web.

UI design

L’UI design permet d’améliorer l’interaction d’un utilisateur avec un produit. Cette réflexion doit être menée pour toutes les étapes de l’utilisation produit, la première d’entre elles étant d’attirer le client. Le designer d’interaction doit alors déterminer quels outils mettre en place pour favoriser l’engagement.

SEO
